## ⛔ MANDATORY RULE — Read Before Any Health/Food Answer

**Before giving ANY health, food, nutrition, medication, exercise, or symptom-related suggestion or answer, you MUST first read the relevant files in this folder.**

Non-negotiable process every time:

1. **Open the actual files below — do not answer from memory, from a conversation summary, or from context you think you already have.** A compaction summary is a lossy copy and drifts from the source. The files in this folder are the single source of truth.
2. **Read [[profile]] first** (the consolidated canonical profile) → then drill into the specific topic pages relevant to the question.
3. Only then answer, grounded in what the files actually say — cite the constraint you're respecting (e.g. "no food/liquid after 6:30 PM", "Allegra/histamine", "no chicken/fish at dinner").

If you are about to give a health answer and you have NOT opened these files this turn, stop and read them first.

**Why this rule exists:** On 2026-07-10 dinner suggestions were given from a session summary instead of the wiki files. The answer was roughly right but the process was wrong — it bypassed the source of truth, which is exactly the failure mode the vault is meant to prevent.

## Hard Constraints (Quick Reference — but still read the files)

- **Dinner:** no chicken or fish — eggs OK as protein. Light, before 6:30 PM.
- **After 6:30 PM:** no food or liquid of any kind (gastric/sleep protection).
- **Sinusitis active:** avoid spicy food (red/green chilli → capsaicin → sloshing), cold food, high-histamine/fermented foods on flare days. On Allegra 120mg.
- **BMI 17–18 (underweight):** calorie-dense choices; target ~2,200–2,400 kcal, 100–120g protein/day.
- **Gastric-prone:** avoid raw legumes/chickpeas/broccoli, excess raw onion at night.
- **Anxiety unmedicated since ~Jan 2026:** food, sleep, movement carry the load — no pharmaceutical backstop.
- **Low BP episodes:** stand up slowly.
