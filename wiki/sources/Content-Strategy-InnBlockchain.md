## Architecture overview

- **7 shared technology pillars** (P1–P7) — Smart Contract Security & Auditing, Protocol Architecture, ZK/MPC/Crypto, Engineering Velocity & Talent, EU Regulatory Compliance, Private & Hybrid Blockchain (FT only), Blockchain ROI & Business Case (FT primary)
- **Track A: Crypto-Native Builders** — 10 segments (A1 RWA primary Q2 2026 → A6 Token Launch active; A7–A10 Watch). Channels: X/Telegram/ETHGlobal/GitHub. ACV $20k–$200k. 1–3 week cycle.
- **Track B: Regulated FinTech** — 8 clusters (B1 WealthTech primary Q2 2026 → B4 WalletTech active; B5–B8 Watch). Channels: LinkedIn ABM / Money20/20 Europe / Sibos Europe / Finextra. ACV $20k–$300k. 60–90 day cycle (+ Carol 2–8 weeks).
- **White-Label Productized Platform motion** runs concurrently across both tracks. Unified offering at $20k+ flat floor. 7 articles total (WL-0 cross-vertical TCO + WL-1 through WL-6 per vertical). Reference Build Hard Gate applies (screenshots required; WL-0 exempt).

## Three sales motions

1. **Security Audit** — review existing smart contract code pre-mainnet. Outbound-led. CTA: *"Request a pre-audit review"* (Track A) / *"Request a regulator-grade audit review"* (Track B).
2. **Development + Engineering** — build the protocol or contracts from scratch. Outbound-led. CTA: *"Scope a custom build"* (Track A — multisig/stablecoin-friendly) / *"Book a discovery call"* (Track B — invoice + procurement).
3. **White-Label Productized Platform** — productized vertical build (contracts + UI + audit) at $20k+. **Buyer owns the codebase outright — not SaaS.** SEO-driven inbound. CTA: *"Book a white-label scoping call"* — one CTA, both ICPs.

Mixing motions in one article's CTA = the most common failure mode per ICP. Sales Motion Tagging is mandatory: `[Audit]` / `[Dev]` / `[White-Label]` / `[Audit + Dev]` / `[Carol]` (Track B only) / `[Cross-Motion]` (rare).
