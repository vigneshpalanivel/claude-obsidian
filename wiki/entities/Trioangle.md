# Trioangle

Software company operating since **2015**. Specializes in launch-ready marketplace clone scripts and custom on-demand platform development. 800+ platforms deployed across 150+ countries.

---

## What They Sell

**Product line** (readymade scripts + super-app platforms):
- Single-vertical clone scripts — Uber, Airbnb, UberEats, TaskRabbit, TikTok, Udemy equivalents
- Gojek-like Super-App — multi-service platform (taxi, delivery, grocery, handyman, car wash, laundry, beauty) controlled from a single admin panel
- Developer Merge — any combination of scripts merged into one branded app

**Service line** (custom development):
- Fully proprietary platforms when clone scripts cannot support the required business logic
- Priced from $10k+ — see `ICP - Service` (separate document)

## Product Tier Routing Logic

```
Does buyer need 1 service only?
  → Single-Vertical Script ($500–$15k)

Do all required services fall within Gojek native scope?
  (taxi, delivery, grocery, car wash, laundry, beauty, handyman, DeliveryAll)
  → Gojek-like Super-App ($8k–$25k)

Any service outside Gojek native scope?
  (Airbnb/Rental, eCommerce, Classifieds, Social/Entertainment)
  → Developer Merge (script price + merge fee)

Fully proprietary / unique business logic?
  → ICP - Service (custom dev, $10k+)
```

## Competitive Positioning

**Core promise**: A live, deployed, App Store-approved business — not code. Trioangle ships the outcome, not the starting point.

**Key differentiators**:
- 800+ deployments since 2015 (proof of production survival)
- Includes free installation, server setup, App Store submission
- 6-month dedicated support post-purchase
- Gojek native scope covers 8 service types via admin toggle — zero dev work

**AI-era positioning**: As vibe coding tools improve, the production deployment gap is the moat. Buyers who spend 4–8 weeks in Cursor/Lovable arrive at Trioangle as [[Trioangle Buyer Personas#Persona 6 — AI Builder Who Hit the Wall|Persona 6]]. Persona 1 (Visionary Velocity) is the most at-risk segment as AI deployment automation matures.
