# Content Repurposing Workflow

Content repurposing is a systematic process for extracting additional value from existing content by transforming it into new formats for different platforms and audiences. One long-form piece can generate 10+ derivative assets.

## Three Repurposing Approaches

**One-to-one** — repackage the same content for a different medium without changing the substance (e.g., blog post → email newsletter).

**One-to-many** — break one piece into multiple smaller assets across platforms (e.g., webinar → clips + quotes + LinkedIn posts + blog summary).

**Content atomization** — create new, inspired-by assets that go beyond the original (e.g., a statistic from a report becomes an original research post).

## Workflow Steps

1. **Audit existing content** — evaluate by topic, format, and performance (engagement, reach, conversions)
2. **Identify repurposing candidates** — high performers and evergreen content first
3. **Set SMART goals** — define what success looks like for each repurposed piece
4. **Choose format and platform** — match the derivative format to the target channel
5. **Produce derivative assets** — adapt structure and length for the platform
6. **Schedule and distribute** — stagger across channels; don't publish all derivatives at once
7. **Track performance** — measure derivative assets separately from the original

## Common Repurposing Paths

- Long-form blog → email newsletter → LinkedIn carousel → short video → quote card
- Webinar → blog recap → podcast episode → clips → slide deck on SlideShare
- Research report → infographic → Twitter/X thread → press release

## Timing Rule

Plan repurposing *during* initial content creation, not after. Build the source asset with derivatives in mind (modular structure, clean quotes, stat callouts).
