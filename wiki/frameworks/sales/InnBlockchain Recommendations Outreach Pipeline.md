# Recommendations Outreach Pipeline

> [!INFO] LinkedIn recommendation outreach framework · cross-phase
> Recommendations are evergreen credibility infrastructure — same outreach pattern applies every phase. Phase 1 ran in SPEED MODE (batched 5–8 requests in one week); Phase 2+ can revert to the slower 4-week stagger from [[Recommendation Request Templates]] § Operational Schedule if no urgency.

## Outreach pipeline

### SPEED MODE (batched · use when urgency requires)

*(Trade-off: recommendations landing in a 7–10 day window is slightly less "earned-looking" than staggered, but acceptable when speed matters. Stagger across 5 working days vs 1 day if even mild smoothing is desired.)*

- [ ] **(Vignesh)** **Day 1–2 (Tue–Wed):** send 2 Category 1 (FinTech client) + 1 Category 5 (engineering hire) requests via LinkedIn DM
- [ ] **(Vignesh)** **Day 3–4 (Thu–Fri):** send 2 Category 2 (crypto-native client) + 1 Category 3 (security peer) + 1 Category 4 (regulator-adjacent) requests
- [ ] **(Vignesh)** **Day 5–7 (next Mon–Wed):** follow up on non-responses (one polite nudge max); write 2–3 reciprocity recommendations for yes-responders

### Standard mode (staggered · default when no urgency)

Per [[Recommendation Request Templates]] § Operational Schedule — 4-week stagger. One request per category per week. Recommended for Phase 2+ when no acute publish gate.

## Activation gates (per phase)

- [ ] **(Vignesh)** **Pre-publish minimum:** 2–3 published (profile-publish gate)
- [ ] **(Vignesh)** **Pre-structured-DM minimum:** 5 published across 3 of 5 categories (gate for structured-outreach activation)
- [ ] **(Vignesh)** **Sasha-grade target:** 7–8 published across all 5 categories (ongoing — can land over weeks)

### Phase 1 overlay

- Phase 1 SPEED MODE target: 3+ by W3 EOD · 5+ before structured DMs activate W5

## Recommender filter (do NOT ask)

- [ ] **(Vignesh)** Family / friends / same surname — visible disqualifier
- [ ] **(Vignesh)** Anyone whose own LinkedIn looks unprofessional
- [ ] **(Vignesh)** Anyone who hasn't done actual paid work
- [ ] **(Vignesh)** People not spoken to in 12+ months
- [ ] **(Vignesh)** Anyone under NDA on the only relevant work
- [ ] **(Vignesh)** Recommenders who would all post within a 48-hour window — stagger

## Why this is a separate doc

Recommendations are not Phase-1-specific — they're credibility infrastructure every phase needs. The 5-category framework + filter + activation gates are stable; only the specific people approached + SPEED-vs-standard cadence shift per phase.
