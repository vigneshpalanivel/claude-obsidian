# Event Execution Playbook

> [!INFO] Per-event execution framework · cross-phase
> Run this checklist for every conference, summit, or roundtable Vignesh attends. Same pattern Phase 1 → Phase 2+. Phase-specific event names (Money20/20 · Sibos · RWA Summit · EU Security Token Summit · DLT Pilot Regime forums) live in the playbook's WS11 § EU events list; this framework covers what to DO at each one.

## Pre-event (4+ weeks ahead)

- [ ] **(Vignesh)** Book speaking slot OR host roundtable (if speaking-slot window open per event-specific lead time)
- [ ] **(Vignesh + SP assist on outbound)** Book meetings 4 weeks in advance (don't just attend — empty calendar = wasted trip)
- [ ] **(Vignesh)** Confirm pre-event collateral ready (vendor risk pack · sample audit report · compliance brief for instant delivery on-request after meetings)

## At event

- [ ] **(Vignesh)** Lead with current-phase regulatory positioning (Phase 1: MiFID II + DLT Pilot framing · never MiCA on RWA)
- [ ] **(Vignesh)** Tag-team SP follow-up — meeting → SP captures next-step in CRM same day
- [ ] **(Vignesh)** Collect names + contexts of high-signal prospects met (vendor evaluation stage · regulatory questions raised · ICP fit)

## Post-event (within 48 hr)

- [ ] **(Vignesh authors · MPs schedule + amplify)** Post-event LinkedIn post with key takeaway (NOT just a photo — substantive observation from a session or conversation)
- [ ] **(MPs)** Tag event organiser (they often reshare)
- [ ] **(MPs)** Link recording / slide deck in first comment (per LinkedIn engagement tactic — external links off the body)
- [ ] **(SP)** Send follow-up DMs to every meeting contact within 48 hr · log in CRM per [[InnBlockchain - Pipeline Stage Exit Criteria]]
- [ ] **(SP)** Deliver promised vendor risk pack / sample audit within 24 hr if any prospect requested it

## Speaking-slot lead time guidance (per event class)

| Event class | Speaking-slot CFP lead time | Action if slot closed |
|---|---|---|
| Tier 1 (Money20/20 · Sibos · RWA Summit · EU Security Token Summit) | 6–12 months pre-event | Attend + book meetings only · pursue next year's CFP |
| Tier 2 (regulatory forums · industry roundtables) | 1–3 months | Often last-minute speaking slots open · pitch directly |
| Tier 3 (community meetups · co-host roundtables) | 2–4 weeks | Self-host: cheaper than attending, higher signal |

## Phase 1 overlay

- Phase 1 events: Money20/20 Europe (W3 attendance only — speaking window closed) · Sibos (Oct 2026 — pursue speaking now) · RWA Summit EU editions (pursue speaking · MiFID II + DLT Pilot framing only) · EU Security Token Summit (pursue speaking) · DLT Pilot Regime / MiFID II tokenization regulatory forums (attend / speak)
- Money20/20 Europe Jun 2027 speaking-slot pursuit (typically Q4 2026 CFP) → Phase 2 playbook
